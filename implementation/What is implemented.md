##What is implemented

We are implementing the Dynamic Discovery part of the Distributed Computing Protocol V1.0 Specification  

Lors du démarrage d'un Smart Calculator, le message HELLO est bien recu par le Compute Engine mais sa réponse (HERE_I_AM) n'est pas reçue par le Smart Caclculator.
Par contre lors du démarrage d'un Compute Engine le message HERE_I_AM est bien recu par le Smart Calculator.
